About This Project

Product Price Comparison is a web-based application designed to help users make smarter shopping decisions by comparing prices of the same product across multiple vendors. Whether you're hunting for the best deal on a smartphone, laptop, or home appliance, this tool simplifies the process by presenting all the options in one place.

Technologies Used

  • Frontend: HTML, CSS, JavaScript
  • Backend: Python (Flask)
  • Database: SQLite
  • Styling: Responsive design with vibrant UI for a smooth user experience
